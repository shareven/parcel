This is a free, open-source, ad-free, offline app that prioritizes simplicity and does not collect any personal information.

This app automatically parses received SMS messages and extracts the address and pickup code information, which can be displayed on a desktop card (dark mode supported).

You can add custom rules to improve the parsing effect.

✅ Supports Taobao and Pinduoduo identity codes.

🔔 Supports monitoring third-party app notifications to automatically save pickup codes—making it much easier to help friends on WeChat pick up their packages.

⚡ Enabling notification access also helps keep the background process alive for real-time widget updates.

🏷️ Supports address categorization; group multiple addresses under one label for easier viewing.

👴 Supports Senior Mode with large fonts for better accessibility.

📱 Adding Home Screen Widgets: Usually found in the "All Widgets" section—look for "Plugins" or "Android Widgets" at the bottom.
